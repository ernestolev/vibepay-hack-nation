# VibePay 💳✨

**Challenge:** Reto 1 - Generative City-Wallet (DSV Gruppe / Sparkassen)
**Tech Stack:** React (Lovable), Cursor AI, Vercel, Tavily API.

## 🚀 Vision
VibePay bridges the gap between traditional banking and the pulse of the city. We use AI to sense the environment and generate dynamic financial incentives that drive foot traffic to local businesses.

## 🛠️ Key Features
- **Context Sensing Layer:** Processes weather, time, and local event data.
- **GenUI Engine:** Generates real-time, emotionally resonant offer widgets.
- **Payone Integration:** Simulated transaction flow for seamless redemption.
- **Privacy-First:** Designed with SLM (Small Language Models) logic in mind for GDPR compliance.
